# Logo-Bild Installation

## Schritt 1: Bild speichern
Bitte speichern Sie Ihr Logo-Bild (der Teppich mit Weltkarte) als:
**images/logo-background.jpg**

## Schritt 2: Optimierung (empfohlen)
- Größe: Mindestens 1920x1080px
- Format: JPG (für beste Performance)
- Qualität: 85% (guter Kompromiss zwischen Qualität und Dateigröße)
- Maximale Dateigröße: 500KB

## Schnell-Anleitung:
1. Ziehen Sie das Bild in den "images" Ordner
2. Benennen Sie es um zu: logo-background.jpg
3. Aktualisieren Sie die index.html Seite im Browser (F5)

## Alternative Namen (falls gewünscht):
Wenn Sie einen anderen Dateinamen verwenden möchten, ändern Sie in:
- css/style.css: Zeile mit `url('../images/logo-background.jpg')`
- css/pages.css: Zeile mit `url('../images/logo-background.jpg')`

## Testen:
Öffnen Sie index.html im Browser - das Bild sollte als Hintergrund im Hero-Bereich erscheinen.

Die Website wurde vollständig an Ihr Logo angepasst mit:
✓ Verbesserte Textschatten für bessere Lesbarkeit
✓ Overlay-Effekte für optimalen Kontrast
✓ Animierte Titel mit goldener Aura
✓ Angepasste Buttons für bessere Sichtbarkeit
✓ Responsive Design für alle Geräte
✓ Parallax-Effekt beim Scrollen (Desktop)
